Scope

What 'scope' is, and how it works in JavaScript

Scope defines what can be accessed by what, in Javascript that means what the variables, functions and objects have access to. Think of scope as a security pass it might let you into all locations or just some depending on your security clearance.
In JavaScript the script can either be local or global. If a variable is declared within a function it is local and can only be accessed within that function. If a variable is declared outside of a function it can be accessed by all functions and scripts that make up the webpage. If a variable is given a value within a function without being declared it is automatically global.
The general rule is only to use global variables if you have a need to just like you would only give full security clearance if it is actually required.
