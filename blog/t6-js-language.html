JavaScript Language

Use case examples

Parentheses ()

Parentheses main use in JavaScript is to group and contain expressions and parameters in functions and structures. For example a parameter is passed to a function which does something with it and returns the result.

var result = doSomething(aThing)

They are also used to group expressions to control the order of execution.

var result = (5+9)/2*4

Some functions have no parameters and in this case, the space between parentheses is blank.

function doSomeThingElse()

Brackets []

Brackets are used to denote an array and either contains its elements or specifies an element by its index number i.e. its position in the array.

myArray = ["a", "b", "c"] and var result = myArray[0] which is "a" as an array is zero indexed.

Braces {}

Braces define the beginning and end of functions and statement blocks such as the for(), if() and function() structures. Curly braces are also used for defining inital values in object declarations.

code block() {
do some stuff
}

Single quotes '' and double quotes ""

There is no meaningful difference between single and double quotes in JavaScript just remember that they have to be used in pairs. They are used to denote a string e.g. myString = "Hello World". However if you are using one form of quote in the string, its a good idea use the other as the literal. e.g. alert('Say "Hello"'); results in Say "Hello"
