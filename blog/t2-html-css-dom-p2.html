A reflection on what was learnt from Udacity lesson 2

10 December 2016

What is a grid based design

Grid based design is where a grid consisting of rows and columns is used as a template on which the position of elements can be determined helping in their placement on the page. The norm is to work from top to bottom and left to right which the eye naturally follows (if your from a western culture).

Whats all the hype about responsive webpages?

They are a response to todays world where there are a large variety of devices with different screen sizes that can access the internet. A responsive webpage will adjust itself automatically according to the size of the screen or more accurately the window the page is in, displaying the content in an appropriate way. This is done by changing the attributes such as size of elements, their position or if they are shown at all.

What is semantic structure?

Semantic structure is where the HTML code is written in such a way that it conveys meaning when read. For example a semantic element would be nav meaning navigation,other examples are footer and article.

Any other points you found interesting and want to reflect on

I found the following points interesting.

  • Use of negative space. Having areas of white space on the page makes it less cluttered and easy to take in the content.
  • The use of Overflow to fit a lot of text into a small area by enabling the scroll feature.
  • Using media queries so that the page is responsive and adapts to the device it is being displayed on.
  • The need to be aware that your site will be viewed on different browsers and different versions and that you can use CSS resetting to account for this.
