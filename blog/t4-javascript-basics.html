The First Steps in JS

How does JavaScript compare to HTML and CSS?

Javascript is a "client-side" programming language. Where HTML provides the content and CSS the style, Javascript provides the ability to "do stuff". It makes a web site interactive and is used to build web applications. Examples would be "pop up" boxes i.e. for user input such as logging in, slideshows, automatic content refresh, search functions, data input and return, calculations basically everything a programming language can do up to full blown applications such as games or ordering your Uber.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow and loops dictate programmatically how the code runs under different conditions or until a certain condition is met. It uses if/else, switch, for and while statements to control what happens and how long for. An example could be the process of waking up:

  • If it is a work day tomorrow - set the alarm for 7 a.m.
  • Else - do not set the alarm
  • While the alarm goes off
  • Switch - how tired am I
  • case1 - awake: Switch off alarm and leap out of bed
  • break
  • case2 - tired: Hit the snooze button and grab another 5
  • break
  • case3 - sick: Switch off alarm and go back to sleep
  • break
  • Default - sleep throuh the alarm and be late for work
Explain the difference between accessing data from arrays and objects.

Data is accessed in an array using the index. So for myArray["Jon", "Bennett", "address"] the address is accessed by myArray[2] (important to note that the indexing starts at 0). Data in objects are accessed by their property name either using dot or bracket notation. So myObject = {firstName: "Jon", lastName: "Bennett", address: "Wellington"} and the data e.g first name, can be accessed by either myObject.firstName or myObject["firstName"].

Explain what functions are and why they are useful.

Functions are blocks of code that perform a task such as a calculation. They take in parameters, perform the task on them and then return a result. They are very useful because it saves having to repeat code for a task you just have to call the function with the necessery parameters.
